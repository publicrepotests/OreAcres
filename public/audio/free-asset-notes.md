# Free Asset Notes

The game currently uses custom project assets plus a built-in WebAudio chiptune loop to avoid licensing risk.

Good free sources to review before importing third-party files:

- Kenney game asset packs: broadly free game assets with simple license terms.
- OpenGameArt: search for CC0 chiptune loops or CC0 pixel UI packs.

Before adding any third-party audio or art, save the source URL and license text here so deployment and investor demos stay clean.
